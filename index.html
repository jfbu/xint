xint repository

  • xint.dtx

    Source for both the documentation and the TeX macro files. If you want to extract the macro files from it, execute etex xint.dtx. If you want then to build the PDF documentation, execute make xint.pdf with the help of provided Makefile, or run latex three times on file xint.tex which was extracted in first step, then execute dvipdfmx xint.dvi. The file xint.pdf will contain both the user manual and the commented source code.
